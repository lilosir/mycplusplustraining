# mycplusplustraining

### On Unix or MacOS

```g++ -o main -std=c++0x my.cpp && ./main```

```g++``` is GUN complier;
```-o main``` is output run file, which has no suffix; 
```-std=c++0x``` speficy new c++ support, such as c++11, c++14
```my.cpp``` is the original file
```&& ./main``` run the output file after compilation
